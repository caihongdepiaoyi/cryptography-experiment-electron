应用简介：
本应用前端使用electron框架 jquery库(js + css + html)开发
Electron 是一个使用 JavaScript, HTML 和 CSS 等 Web 技术创建原生程序的框架
算法使用C语言开发,可以实现文件加解密等功能
使用方法：
  • 1.加密时点击选择文件按钮，然后点击开始加密，然后等待加密过程完成即可
  • 2.加密完成后会在result目录下生成密文文件和公钥私钥
  • 3.密文文件和公钥私钥并分别命名为Out.Bin，public_key.pem，private_key.pem
  • 4.解密时点击选择文件按钮，然后点击加载密钥，接着点击加载密钥，然后点击开始解密，最后等待解密过程完成即可
  • 5.解密完成后点击保存到文件，将解密后的信息另存于指定文件
注意事项：
  • 1.请勿手动删除或修改result目录下的公钥与私钥文件内容，否则会导致无法解密
  • 2.加密一些较大文件时可能会花费较长时间请耐心等待
